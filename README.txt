The main CLI commands to be executed on first start of this application are automated inside of .sh scripts.
You can modify them at will when needed. 



To tell Flask we want this app to run:

>>> source set_flask_app.sh



To tell Flask to run a development server:

>>> source set_flask_dev_env.sh



To tell Flask to run a production server (discouraged):

>>> source set_flask_prod_env.sh
